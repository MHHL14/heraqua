2. Productie in Magento 2 (v2c module)

Magento-module serveert widget + product-data. Node.js AI-backend blijft een aparte service (externe host).

Magento-module Serveert widget-assets + /adviseur/products.json, admin-UI voor config & field-mapping.
Node-backend Aparte service (bv. Railway/Fly.io). Praat met Anthropic. Beveiligd met shared token + CORS + rate-limit.
Cron + Observer Cron rebuildt products.json cache (1×/dag). Observer markt 'dirty' bij product-save; volgende cron-tick rebuildt.
Widget-embed Drie manieren: CMS widget (drag & drop), layout XML (developer), auto-embed op categoriepagina's (config-toggle).
